# Usage

- Request Url Examples:
```
111.111.111.111/talk
111.111.111.111:8080/talk
example.com:8080/talk
example.com/talk
```

- Post
```
{"lang":"en", "message":"Message", "name":"AI Name"}
```

- Response
```
{'id': response id , 'message': 'response message'}
```
